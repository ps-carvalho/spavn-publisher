# 👤 Profile Settings

Your profile settings allow you to manage your personal information, update your password, and control your account security. This guide covers everything you need to know about managing your own account.

---

## 🔐 Accessing Your Profile Settings

To access your profile settings:

1. Log in to Publisher CMS
2. Click on your **avatar or initials** in the top right corner
3. Select **Profile Settings** from the dropdown menu
4. You'll see your profile page with editable fields

### Profile Settings Overview

| Section | What You Can Do |
|---------|-----------------|
| **Profile Information** | Update name, email, bio, avatar |
| **Password** | Change your current password |
| **Sessions** | View and manage active sessions |
| **Preferences** | Customize your experience |

---

## ✏️ Updating Personal Information

### Editable Profile Fields

| Field | Description | Requirements |
|-------|-------------|--------------|
| **Name** | Your display name shown throughout the CMS | Required, visible to other users |
| **Email** | Your login email address | Valid email, must be unique |
| **Bio** | Short biography or description | Optional, visible on profile |
| **Avatar** | Profile picture | Optional, uploaded via media library |

### How to Update Your Profile

1. Navigate to **Profile Settings**
2. Modify the fields you want to change
3. Click **Save Changes**
4. You'll see a confirmation message

### Changing Your Email

> ⚠️ **Important**: Changing your email also changes your login credentials.

1. Enter your new email address
2. Click **Save Changes**
3. You may receive a verification email at the new address
4. Click the verification link to confirm the change

## 🔑 Changing Your Password

### How to Change Your Password

1. Navigate to **Profile Settings**
2. Scroll to the **Password** section
3. Enter your **current password**
4. Enter your **new password**
5. Confirm your **new password**
6. Click **Update Password**

### Password Requirements

| Requirement | Details |
|-------------|---------|
| Minimum length | 8 characters |
| Maximum length | 128 characters |
| Recommended | 12+ characters |

### Password Strength Guidelines

| Strength | Example | Characteristics |
|----------|---------|-----------------|
| ❌ **Weak** | `password123` | Common words, predictable patterns |
| ⚠️ **Fair** | `Blue2024!` | Some complexity but guessable |
| ✅ **Strong** | `K9$mPxL2@nQv` | Random mix of characters |
| 💪 **Excellent** | `Tr0ub4dor&3-Horse!Battery` | Long passphrase with symbols |

### Creating a Strong Password

Use these tips to create a secure password:

| Tip | Example |
|-----|---------|
| Use a passphrase | `correct-horse-battery-staple` |
| Mix character types | `Upp3rC4se&L0w3rC4se!` |
| Add numbers and symbols | `MyP@ssw0rd#2024` |
| Avoid personal info | ❌ Don't use: birthday, name, pet's name |
| Use unique passwords | Different password for each service |

## 📱 Session Management

Publisher CMS tracks your active login sessions. This helps you monitor where you're logged in and identify any unauthorized access.

### Viewing Active Sessions

1. Navigate to **Profile Settings**
2. Scroll to the **Sessions** or **Security** section
3. You'll see a list of active sessions with:
   - Device type (Desktop, Mobile, Tablet)
   - Browser information
   - IP address
   - Last activity timestamp
   - Location (approximate)

### Session Information

| Field | Description |
|-------|-------------|
| **Device** | Type of device used |
| **Browser** | Browser name and version |
| **IP Address** | Network address used |
| **Location** | Approximate geographic location |
| **Last Active** | Most recent activity on this session |
| **Current** | Indicates if this is your current session |

### Session Security Settings

| Setting | Value | Description |
|---------|-------|-------------|
| **Cookie Name** | `publisher-session` | Session cookie identifier |
| **Token Expiry** | 7 days | Session automatically expires after 7 days |
| **Password Hashing** | bcrypt (12 rounds) | Industry-standard password security |

## 🚪 Logging Out from All Devices

If you suspect unauthorized access or want to secure your account:

### How to Log Out Everywhere

1. Navigate to **Profile Settings**
2. Scroll to the **Sessions** section
3. Click **Log Out All Other Sessions** or **Revoke All Sessions**
4. Confirm the action
5. You may need to log in again

### When to Use This Feature

| Scenario | Action |
|----------|--------|
| Lost or stolen device | Log out all sessions immediately |
| Suspicious activity | Log out all, change password |
| Shared computer used | Log out all sessions |
| Regular security practice | Log out all sessions monthly |

> 🔐 **Best Practice**: After logging out all sessions, change your password for added security.

## 🛡️ Security Best Practices

### Password Security

| Do ✅ | Don't ❌ |
|-------|----------|
| Use unique passwords for each service | Reuse passwords across sites |
| Use a password manager | Write passwords on paper |
| Change password if you suspect compromise | Share passwords with others |
| Use 2FA when available | Use easily guessable info |

### Account Security Checklist

Review these items regularly:

- [ ] Password is strong and unique
- [ ] Email address is current and secure
- [ ] No unfamiliar sessions in your session list
- [ ] Recovery options are up to date
- [ ] You recognize all active sessions

### Recognizing Suspicious Activity

Watch for these warning signs:

| Warning Sign | What It Means |
|--------------|---------------|
| Unfamiliar sessions | Someone may have accessed your account |
| Login emails you didn't initiate | Possible unauthorized login attempt |
| Settings changed without your action | Account may be compromised |
| Content you didn't create | Someone using your account |

### If Your Account Is Compromised

1. **Immediately** change your password
2. Log out all other sessions
3. Review recent activity for unauthorized changes
4. Contact your administrator
5. Check for any changed settings

## 🔄 Password Reset (Forgot Password)

If you forget your password and can't log in:

### Self-Service Password Reset

1. Go to the Publisher CMS login page
2. Click **Forgot Password?** below the login form
3. Enter your registered email address
4. Check your email for a reset link
5. Click the link (expires after a limited time)
6. Create a new strong password
7. Log in with your new password

### Reset Email Not Arriving?

| Check | Solution |
|-------|----------|
| Spam folder | Check junk/spam folders |
| Correct email | Verify you used the right email address |
| Email delays | Wait 5-10 minutes |
| Contact admin | Ask administrator to reset your password |

## 💡 Tips for Managing Your Account

> 🔐 **Regular password updates**: Consider changing your password every 3-6 months.

> 📱 **Review sessions weekly**: Check your active sessions to catch any unauthorized access early.

> 📧 **Keep email current**: Ensure your email is always accessible for password resets.

> 🚨 **Report issues**: If you notice anything suspicious, report it to your administrator immediately.

> 🔒 **Log out on shared devices**: Always log out when using public or shared computers.
